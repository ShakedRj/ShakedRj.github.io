Machine Learning Analysis

I performed five diffrenet ML algorithms on the heart failure dataset in order to determine which one will be the most accurate in predicting death in heart failure paitents.
This is a classification probelm so I used SVM, Decision Tree classifier, Gradient Boosting classifier, Random Forest classifier and KNN algorithms.
I performed ML calculations and data manipulation in Python and Tableau, and exported the results from python so I can visualize it in Tableau.

Algorithm: Accuracy Score:
KNN 0.62
SVM 0.76
Decision Tree classifier 0.77
Random Forest classifier 0.84
Gradient Boosting classifier 0.86
